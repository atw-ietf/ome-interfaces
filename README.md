# Registration & Query Interfaces for DRIP Identity Management Entities (DIMEs)

This is the working area for the IETF Internet-Draft, "Registration & Query Interfaces for DRIP Identity Management Entities (DIMEs)".

## Building the Draft

Formatted text, XML and HTML versions of the draft can be built using `make`.

```sh
make clean && make
```

This requires that you have the necessary software installed. See [the instructions](https://github.com/martinthomson/i-d-template/blob/master/doc/SETUP.md).
